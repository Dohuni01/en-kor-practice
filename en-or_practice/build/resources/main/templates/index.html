영어 퀴즈 & 플래시카드 메이커

한 줄에 영어 + 한국어 뜻만 적으면 바로 퀴즈와 플래시카드 학습을 할 수 있습니다.

퀴즈 자동 생성 플래시카드 학습 오답 다시 풀기 로컬 저장

STEP 1

문장 입력

한 줄에 하나씩 입력

영어 문장 뒤에 한국어 뜻을 붙여서 입력하세요. 서버는 첫 번째 한글부터 한국어 뜻으로 인식합니다. 영어 정답에 여러 표현을 허용하고 싶으면 | 또는 [ ]를 써도 됩니다.

문장을 입력하면 자동으로 형식을 확인합니다.

STEP 2

입력 미리보기

0 인식한 카드
0 형식 확인 필요

입력하면 여기서 바로 카드 미리보기를 볼 수 있습니다.

STEP 3

퀴즈 설정

엔터로 정답 확인, 채점 뒤에는 엔터로 다음 문제로 넘어갑니다.

STEP 4

단어 카드 학습

카드 한 장씩 넘기면서 외울 수 있게 만들었습니다. 기본은 한국어 먼저이고, 카드를 누르면 영어가 보입니다.

카드 0 / 0 한국어 먼저

현재 카드는 앞면이 보이는 상태입니다.

문제 0 / 0 점수 0
영어 → 한국어
문제를 보고 답을 입력하세요.

STEP 5

결과 정리

지금 버전은 브라우저에서 바로 돌 수 있도록 프론트는 가볍게 두고, 입력 파싱은 스프링에서 처리하게 만든 구조입니다. 나중에 원하면 로그인, DB 저장, 오답노트까지 붙이기 쉽습니다.
