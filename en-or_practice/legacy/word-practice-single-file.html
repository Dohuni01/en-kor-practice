영어 퀴즈 & 플래시카드 메이커

한 줄에 하나씩 영어 + 한국어 뜻을 넣으면, 퀴즈와 플래시카드 학습을 바로 할 수 있는 암기용 사이트입니다.
예시 입력: Organizing documents 서류를 정돈하다

1. 문장 입력

영어 문장 뒤에 한국어 뜻을 붙여서 입력하세요. 사이트는 첫 번째 한글부터 한국어 뜻으로 인식합니다.
영어 정답에 여러 표현을 허용하고 싶다면 | 또는 대괄호를 사용할 수 있습니다.
예: Plugging[unplugging] a cord 코드를 꽂다

입력한 문장을 확인하는 중입니다.

2. 퀴즈 설정

엔터를 누르면 정답 확인, 이미 채점된 문제에서는 엔터로 다음 문제로 넘어갑니다.

3. 단어 학습 카드

Quizlet처럼 카드 한 장씩 보면서 외울 수 있습니다. 기본은 한국어 먼저이고, 카드를 누르면 영어로 뒤집힙니다.

카드 0 / 0 한국어 먼저

현재 카드는 앞면이 보이는 상태입니다.

문제 0 / 0 점수 0
영어 → 한국어
문제를 보고 답을 입력하세요.

4. 결과

이 파일은 브라우저에서 바로 열 수 있는 단일 HTML 버전입니다. 지금은 퀴즈와 플래시카드 학습이 함께 들어 있고, 나중에 원하면 오답노트 저장 / 객관식 / 로그인 / 배포용 버전까지 확장할 수 있습니다.
